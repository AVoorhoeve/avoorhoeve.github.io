    Welk bedrijf heb ik gekozen?

    CD Project Red

    Wat voor projecten doet CD Projekt Red?

    Het is een gamedevelopment studio die onder andere The Witcher games en Cyberpunk 2077 hebben gemaakt.

    Welke IT technologie werkt CD Projekt Red mee?

    Zij gebruiken Unreal Engine 5 om hun games te developen.

    Hoeveel werknemers heeft CD Projekt Red?

    Het bedrijf heeft nu 859 werknemers.

    Wat vertellen ze je over de cultuur van CD Projekt Red?

    Het is een Pools bedrijf dat in 2002 in Warschau is opgericht.

    Is CD Projekt Red Internationaal actief?

    Ja, hun games worden wereldwijd tientallen miljoenen keren verkocht.
